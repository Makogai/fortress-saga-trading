# Album assets (bundled)

Place one folder per album here, named with **dashes** (e.g. `equipment-album`, `hero-album-i`).

- **cover.png** — album cover
- **01.png … 10.png** — card images in game order

These files are bundled by Vite and used on the Albums shelf and inside each album. Missing card images show the album cover as placeholder.
